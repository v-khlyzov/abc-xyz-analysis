# ABC-XYZ анализ ассортимента для определения стратегии управления запасами
**Суть задачи:** Показать, как на основании истории продаж разделить ассортимент по приоритетам и определить разные стратегии управления запасами.

**Данные:** Использован искусственный набор данных, имитирующий продажи товаров за несколько недель.

**Методика расчета:**
- подготовка данных;
- расчет оборота;
- расчет доли;
- ABC-классификация по кумулятивной доле;
- расчет коэффициента вариации;
- XYZ-классификация;
- построение итоговой матрицы.

## Выводы
| Категория | Решение                                          |
| --------- | ------------------------------------------------ |
| AX        | Постоянное наличие, ежедневный контроль          |
| AY        | Регулярный мониторинг, небольшой страховой запас |
| AZ        | Индивидуальное планирование закупок              |
| BX        | Плановое пополнение                              |
| BY        | Контроль по графику                              |
| BZ        | Анализ причин нестабильности                     |
| CX        | Минимальный запас                                |
| CY        | Заказывать по потребности                        |
| CZ        | Рассмотреть вывод из ассортимента                |
